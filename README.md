## Assignment Overview

# Welcome to the Web Application version of Assignment 1: Response Spectrum Analysis of Earthquake Ground Motion Records

a. Generate Ground Acceleration Time History from the selected earthquake record.

b. Conduct Fast Fourier Transform (FFT) analysis to identify dominant frequencies.

c. Calculate values of Spectral Displacements, Spectral Velocities, and Spectral Accelerations for a range of natural periods by solving the SDOF equation of motion using the Newmark Method.

d. Generate and plot the Response Spectrum Curve based on the calculated values.

The ground motion records were provided to us as .txt files.
